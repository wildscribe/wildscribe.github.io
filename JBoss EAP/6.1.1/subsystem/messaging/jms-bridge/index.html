A JMS bridge instance.

Attributes (22)

  • add-messageID-in-header If true, then the original message's message ID will be appended in the message sent to the destination in the header HORNETQ_BRIDGE_MSG_ID_LIST. If the message is bridged more than once, each message ID will be appended.
    • Attribute Value
    Default Value false
    Type BOOLEAN
    Nillable true
    Expressions Allowed true

    Nillable true
    Expressions Allowed true

    • Attribute Value
    Type LONG
    Nillable false
    Expressions Allowed true
  • max-batch-size The maximum number of messages to consume from the source destination before sending them in a batch to the target destination. Its value must >= 1.
    • Attribute Value
    Type INT
    Nillable false
    Expressions Allowed true
    Min 0
    Max 2,147,483,647
