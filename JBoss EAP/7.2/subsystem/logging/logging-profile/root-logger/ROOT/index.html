Defines the root logger for this log context.

Provided capabilities(1)

Name Dynamic Other provider points
org.wildfly.logging.logger true /subsystem=logging/logger=*
/subsystem=logging/root-logger=ROOT
/subsystem=logging/logging-profile=*/logger=*

Attributes (3)

  • filter-spec A filter expression value to define a filter. Example for a filter that does not match a pattern: not(match("JBAS.*"))
    • Attribute Value
    Type STRING
    Nillable true
    Expressions Allowed true
    Storage configuration
    Access Type read-write
    Restart Required no-services
  • handlers The handlers associated with the root logger.
    • Attribute Value
    Type LIST
    Nillable true
    Expressions Allowed false
    Storage configuration
    Access Type read-write
    Restart Required no-services
  • level The log level specifying which message levels will be logged by the root logger. Message levels lower than this value will be discarded.
    • Attribute Value
    Default Value ALL
    Type STRING
    Nillable true
    Expressions Allowed true
    Storage configuration
    Access Type read-write
    Restart Required no-services

Operations (9)

  • add Adds the root logger.
    Request Parameter Type Required Expressions Allowed Default value Description
    filter-spec STRING false true A filter expression value to define a filter. Example for a filter that does not match a pattern: not(match("JBAS.*"))
    handlers LIST false false The handlers associated with the root logger.
    level STRING false true ALL The log level specifying which message levels will be logged by the root logger. Message levels lower than this value will be discarded.
  • add-handler Adds a handler to the logger.
    Request Parameter Type Required Expressions Allowed Default value Description
    name STRING false false The name of the handler to add.
  • change-root-log-level Change the root logger level.
    Deprecated Since 1.2.0
    Use the write-attribute operation.
    Request Parameter Type Required Expressions Allowed Default value Description
    level STRING false true ALL The level to set the root logger to.
  • remove Remove the root logger.
  • remove-handler Removes a handler from the logger.
    Request Parameter Type Required Expressions Allowed Default value Description
    name STRING false false The name of the logger to remove.
  • remove-root-logger Remove the root logger.
    Deprecated Since 1.2.0
    Use the remove operation.
  • root-logger-assign-handler Assign a Handler to the root logger.
    Deprecated Since 1.2.0
    Use the add-handler operation.
    Request Parameter Type Required Expressions Allowed Default value Description
    name STRING false false The name of the handler to add.
  • root-logger-unassign-handler Unassign a Handler from the root logger.
    Deprecated Since 1.2.0
    Use the remove-handler operation.
    Request Parameter Type Required Expressions Allowed Default value Description
    name STRING false false The name of the handler to remove.
  • set-root-logger Same as add operation
    Deprecated Since 1.2.0
    Use the add operation.
