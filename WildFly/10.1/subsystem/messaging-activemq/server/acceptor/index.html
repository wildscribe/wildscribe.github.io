  4. acceptor

An acceptor defines a way in which connections can be made to the ActiveMQ server.

Attributes (4)

  • factory-class Class name of the factory class that can instantiate the acceptor.
    • Attribute Value
    Type STRING
    Nillable false
    Expressions Allowed false
    Storage configuration
    Access Type read-write
    Restart Required all-services
  • params A key-value pair understood by the acceptor factory-class and used to configure it.
    • Attribute Value
    Type OBJECT
    Nillable true
    Expressions Allowed true
    Storage configuration
    Access Type read-write
    Restart Required no-services
  • socket-binding The socket binding that the acceptor will use to accept connections.
    • Attribute Value
    Type STRING
    Nillable true
    Expressions Allowed false
    Storage configuration
    Access Type read-write
    Restart Required all-services
  • started Whether this acceptor is started.
    • Attribute Value
    Type BOOLEAN
    Nillable false
    Expressions Allowed false
    Storage runtime
    Access Type read-only
