  1. WildFly 10.1
  2. subsystem=modcluster

The configuration of the modcluster subsystem.

Children (1)

  • mod-cluster-config Defines configuration resource
    • configuration The common modcluster configuration.

Operations (15)

  • add Operation adding the modcluster subsystem.
  • add-proxy Add the proxy to the list of httpd connected to the node.
    Request Parameter Type Required Expressions Allowed Default value Description
    host STRING true false host of the proxy
    port INT true false port of the proxy
  • disable Tell Apache httpd that all contexts of the node can't process new requests.
  • disable-context Tell Apache httpd that the context can't process new requests.
    Request Parameter Type Required Expressions Allowed Default value Description
    virtualhost STRING true false virtual host
    context STRING true false context
  • enable Tell Apache httpd that all contexts of the node are ready receive requests.
  • enable-context Tell Apache httpd that the context is ready receive requests.
    Request Parameter Type Required Expressions Allowed Default value Description
    virtualhost STRING true false Virtual host
    context STRING true false Context
  • list-proxies Read the list of httpd connected to the node.

    Reply properties

    type
    LIST
    Value Type
    STRING
  • read-proxies-configuration Send a DUMP to httpd and display the result.

    Reply properties

    type
    LIST
    Value Type
    STRING
  • read-proxies-info Send an INFO to httpd and display the result.

    Reply properties

    type
    LIST
    Value Type
    STRING
  • refresh Refresh the node sending a new CONFIG message to Apache httpd.
  • remove Operation removing the modcluster subsystem.
  • remove-proxy Remove the proxy to the list of httpd connected to the node.
    Request Parameter Type Required Expressions Allowed Default value Description
    host STRING true false host of the proxy
    port INT true false port of the proxy
  • reset Reset the node connection to Apache httpd.
  • stop Tell Apache httpd that all contexts of the node can't process requests.
    Request Parameter Type Required Expressions Allowed Default value Description
    waittime INT false false 10 wait timeout
  • stop-context Tell Apache httpd that the context can't process requests.
    Request Parameter Type Required Expressions Allowed Default value Description
    virtualhost STRING true false Virtual host
    context STRING true false Context
    waittime INT false false 10 wait timeout
