  3. ldap-connection
Deprecated Since 1.7.0
The ldap-connection configuration is deprecated and may be removed or moved in future versions.

A connection factory that can be used by a security realm to access an LDAP server as a source of authentication and authorization information.

Children (1)

  • property A custom property to use when establishing the LDAP connection.
