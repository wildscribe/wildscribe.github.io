  2. subsystem=transactions

The configuration of the transaction subsystem.

Children (2)

  • commit-markable-resource a CMR resource (i.e. a local resource that can reliably participate in an XA transaction)
  • log-store Representation of the transaction logging storage mechanism.
    • log-store Representation of the transaction logging storage mechanism.
