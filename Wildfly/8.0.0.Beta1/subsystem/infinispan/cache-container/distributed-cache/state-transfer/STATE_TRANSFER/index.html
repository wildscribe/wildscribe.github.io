  5. state-transfer=STATE_TRANSFER

The state transfer configuration for distribution and replicated caches.

Attributes (3)

  • chunk-size The size, in bytes, in which to batch the transfer of cache entries.
    • Attribute Value
    Default Value 10000
    Type INT
    Nillable true
    Expressions Allowed true
  • enabled If enabled, this will cause the cache to ask neighboring caches for state when it starts up, so the cache starts 'warm', although it will impact startup time.
    • Attribute Value
    Default Value true
    Type BOOLEAN
    Nillable true
    Expressions Allowed true
  • timeout The maximum amount of time (ms) to wait for state from neighboring caches, before throwing an exception and aborting startup.
    • Attribute Value
    Default Value 60000
    Type LONG
    Nillable true
    Expressions Allowed true
